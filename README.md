# EZ Baccarat Wizard

EZバカラ(ノーコミッションバカラ)専用のセッション記録・資金管理PWA。韓国カジノでの個人利用を想定(基準通貨KRW・JPY常時併記)。スマホ縦画面・片手操作前提、完全ローカル動作(IndexedDB)でオフライン対応。

**公開URL**: https://ezb-wizard.github.io/

## 主な機能

- **記録**: 1ハンド3タップ以内のクイック登録、見(観戦)記録、Undo・履歴編集
- **資金管理**: 推奨ベット額(資金の1〜3%)、ストップロス/テイクプロフィット全画面アラート
- **カジノプリセット**: PARADISE(仁川)/ INSPIRE(仁川)の配当プリセット切替+カスタム編集
- **理論値**: 8デッキ完全列挙による出現率・控除率(サイドベットは設定配当から動的算出)
- **統計**: 大路(ビッグロード)、資金推移、実績vs理論の収束チャート
- **検証**: 戦略シミュレーター(モンテカルロ、Web Worker実行)
- **為替**: JPY/KRWレート自動取得(オフライン時はキャッシュ+取得時点表示)

> 本アプリは記録・資金管理・理論値の可視化のみを目的とし、結果の予測や勝利を保証する機能はありません。バカラの各ハンドは独立試行であり、控除率は固定です。

## 開発

```bash
npm install
npm run dev      # 開発サーバー
npm test         # ユニットテスト(精算エンジン・確率テーブル)
npm run build    # 型チェック+本番ビルド(PWA生成込み)
```

`main` ブランチへのプッシュで GitHub Actions がテスト→ビルド→GitHub Pages デプロイを自動実行します。

## 技術構成

React 19 / TypeScript / Vite / Tailwind CSS v4 / Dexie.js(IndexedDB)/ vite-plugin-pwa
